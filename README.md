### Create a new repository on the command line
``` sh
touch README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/ouchliu/ytrey.git
git push -u origin master
```

### Push an existing repository from the command line
``` sh
git remote add origin https://github.com/ouchliu/ytrey.git
git push -u origin master
```
